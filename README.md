Desarrollar un sistema robótico automatizado que utilice un DOBOT Magician para cortar y separar las manos de un racimo de banano orito (racimo pequeño, adecuado al tamaño y capacidad del robot). El sistema usará una cámara para localizar cada mano mediante visión artificial, y un end effector diseñado específicamente para este proyecto, que combine capacidades de corte y sujeción para realizar el corte en el punto de unión y depositar cada mano cortada en una zona de recolección

### Descripcion
Este proyecto busca desarrollar un sistema robotico que permita automatizar el corte de manos de banano para aumentar la productividad y rapidez del proceso. Para el primer prototipo se usara el DOBOT Magician, y una variedad de banano que se adapte a las capacidades del robot, como lo es el banano orito (Nombre cientifico: Musa acuminata (AAA Group) 'Orito').

### Sistema
El DOBOT Magician contará con una cámara para localizar y reconocer cada mano de banano. Luego de identificar las manos del racimo el robot se encargará de cortar las manos de banano y colocarlas en una zona de recolección. Estos dos procesos se realizarán mediante un efector final diseñado desde cero para este proyecto.
